    C程序设计基础及实验¶

    这是大一上唯一与计算机相关的课程，学分也挺高的，由于我之前并无编程基础，我学得比较认真，最后也取得了一个满意的成绩。

    我的老师是陶煜波老师。陶老师绝对是一位好老师，平时十分认真负责，点名会提前通知，作业没交会提醒你补交；经常会讲一些课本上没有的拓展知识，并且也挺捞的。但缺点是对零基础的同学并不友好，如果不提前预习的话上课会很吃力，并且老师的作业和小测是出了名的多且难，这未必是坏事但确实让人有点头疼。

    这门课的计分方式是平时分占65%，实验考占15%，期末理论考占20%，存在双斩杀线。想拿高分并不是什么难事，但这应该是大多数同学包括我的第一门编程课，可能会有与之前的学习很不一样的感觉。我采取的学习方式是一种“半自学”，在课前我会根据老师发的PPT预习知识，课上听老师补充一些。作业一定要认真独立完成，实在不会了再去上网搜或者去求助大模型，打好编程基础需要一定的代码量，想学好C语言实践是必不可少的。千万不要形成了对大模型的路径依赖，不然后期很容易陷入惰于写代码的恶性循环。

    平时分主要由作业和小测组成，只要达到一定线就可以拿满分，其实认真做了分就不会低。为了确保作业分能拿满，每次做完我们寝室都会对答案；小测确实很有难度，但是只要平时认真学了写作业，考前再突击一下也能拿一个不错的分数。实验考比较容易，只要你平时认真写作业会debug就很容易满分，注意要仔细读题，训练讨论各种情况自测的能力。至于理论考，题目有点恶心，但也确实不难，主要是理清概念，平时把自己易错的记录下来，期末做几套历年卷拟合一下，考个高分很容易。
